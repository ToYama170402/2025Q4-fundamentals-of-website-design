# 2025Q4 Webサイト設計基礎

このレポジトリは2025年第4Q開講の[Webサイト設計基礎](https://eduweb.sta.kanazawa-u.ac.jp//Portal/Public/Syllabus/SyllabusSearchStart.aspx?lct_year=2025&fac_cd=55&lct_no=23014&je_cd=1)の個人用演習レポジトリです。

## 環境構築

### 必要ツール

- [Git](https://git-scm.com/)
- [Visual Studio Code](https://code.visualstudio.com/)
- [Docker](https://www.docker.com)

### 手順

1. `git clone https://github.com/ToYama170402/2025Q4-fundamentals-of-website-design.git`
2. `cd 2025Q4-fundamentals-of-website-design`
3. `docker compose up`
